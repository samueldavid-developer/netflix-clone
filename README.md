# Clon de Netflix - Desarrollo con React y TypeScript

Este proyecto es una aplicación funcional inspirada en la interfaz de Netflix. El objetivo principal fue poner en práctica el manejo de estados complejos en React, la integración de servicios externos y el uso de TypeScript para escribir un código más robusto y fácil de mantener.

## Enlace del proyecto
El sitio se encuentra desplegado y listo para ser probado en el siguiente enlace:
https://samueldavid-developer.github.io/netflix-clone/

## Funcionalidades alcanzadas
En esta aplicación logré implementar una navegación dinámica que consume información real de la base de datos de TMDB. Los usuarios pueden explorar películas por categorías, realizar búsquedas personalizadas y visualizar los avances oficiales a través de un reproductor de YouTube integrado. También puse especial atención en que la interfaz fuera fluida y se adaptara correctamente a diferentes tamaños de pantalla.

## Tecnologías y herramientas
Para la construcción de este proyecto utilicé:
* React 19 para la arquitectura de componentes.
* TypeScript para asegurar la integridad de los datos de la API.
* Axios para la comunicación con el servidor de TMDB.
* CSS nativo para lograr los efectos de desplazamiento y el diseño visual.
* Vite como herramienta de compilación y optimización.

## Cómo ejecutarlo en tu equipo
Si deseas revisar el código o realizar pruebas locales, puedes seguir estos pasos:

1. Clona el repositorio en tu máquina.
2. Ejecuta "npm install" en la terminal para descargar las dependencias necesarias.
3. Inicia el entorno de desarrollo con el comando "npm run dev".
